Microsoft Windows


Microsoft windows es el sistema operativo desarrollado para todos los equipos de la empresa, los cuales han ido actualizándose progresivamente.


A continuación daremos a conocer sus productos más importantes:


Sistemas operativos para PC


Es la plataforma digital con la que trabaja un computador, que contiene una variedad de programas que cumplen funciones específicas y controlan los elementos del hardware. En la plataforma los programas se ejecutan, y su interfaz variará de acuerdo a la compañía que lo haya desarrollado. Dichos programas pueden ser adquiridos para este sistema en su microsoft store para varias versiones de windows.


Existe una importante cantidad de sistemas operativos para computadores, tanto de grandes compañías como la de microsoft, como otros independientes. Entre ellos se destacan: Windows, Mac OS, Unix, Solaris, Linux, Ubuntu, Wave OS, entre otros.


El desarrollado por microsoft para PC es el windows, lanzado en 1985 en su versión 1.0, pero más bien como una interfaz gráfica complementaria al sistema MS-DOS. Luego, entre sus actualizaciones más conocidas se encontraron Windows 95, que sustituyó por primera vez a MS-DOS; windows 98; windows XP; windows 7; windows 8; y windows 10.


Sistemas operativos móviles


Windows mobile: es un sistema operativo basado en la tecnología microsoft windows CE diseñado específicamente para dispositivos móviles como Pocket PC (PPC), Smartphones y otros dispositivos de media portátil. Está estrechamente vinculado a otros productos de la misma marca y cuenta con una interfaz gráfica de excelente calidad y similar a las versiones de escritorio de windows, por lo que el entorno de trabajo es muy parecido al del hogar u oficina. Actualmente ha sido opacado por el iPhone de Apple y Android. Por ello, la empresa lo descontinuó para crear Windows Phone.


Windows phone: se trata del sistema operativo que reemplazó a windows mobile a partir del año 2010. Microsoft decidió realizar un cambio completo en este nuevo sistema operativo con respecto al otro; no solo se cambió el nombre, sino que se desarrolló desde cero, presentando una interfaz completamente nueva, mejor comportamiento y un mayor control sobre las plataformas de hardware que lo ejecutan, con el propósito de volver a ser competitivo en el mundo de los móviles. Sin embargo, en el 2015, la compañía decidió retirar del mercado este sistema por la segmentación de sus sistemas operativos.


Windows 10 mobile: fue el sistema operativo que le sucedió al windows phone en su versión 8.1, que fue desarrollado para los dispositivos de telefonía celular y tabletas, pero que en el año 2017 cesó de desarrollarla por su baja demanda y actividad en el mercado. Sin embargo, subsecuentemente ha desarrollado aplicaciones con compatibilidad para dispositivos con los sistemas operativos móviles con Android e iOS.
