Microsoft Office


Es el grupo de programas para labores domésticas y de oficina, que incluyen softwares para la creación y edición de textos, presentaciones, procesamiento de datos, entre otras labores que permiten automatizar procesos.


Este paquete fue creado a finales de los ochenta por la compañía originalmente con PowerPoint, Word y Excel, cada uno de los cuales serán explicados más adelante. Microsoft 365 es el servicio a través del cual se puede suscribir por un año al paquete de programas de office, en la tienda de microsoft online.


Word


Es un programa que permite el proceso de texto de Office, siendo el procesador más utilizado en el mundo y puede soportar varios formatos. Entre estos se encuentran el DOC, que es el propio del programa, siendo su extensión .doc y tiene un formato que permite una mayor comprensión que es el .docx; el formato RTF, que permite que un archivo de texto con su extensión .rtf pueda ser abierto en cualquier versión de Word.


Este programa tiene la característica de corrector de ortografía, sinónimos y una amplia gama de opciones para dar formato al texto. Desde el año 2016 el programa permite exportar o guardar los documentos en formato PDF.


Excel


Es un software que permite desarrollar trabajos contables y financieros gracias a sus funciones, creadas especialmente para elaborar hojas de cálculo. Actualmente posee una central formada por columnas y filas, lo que dan paso a la formación de las denominadas celdas, que contará con una dirección específica asignada, determinado por la columna y fila a la que pertenece. En dicha celdas es posible colocar datos numéricos y alfanuméricos.


Una herramienta muy útil que provee Excel es la de realizar cálculos aritméticos mediante el uso de fórmulas, seguido por regla de un signo igual (=). Así como estas son muchas las herramientas que este programa te ofrece, por lo que en la actualidad se ha convertido en uno de los programas favoritos para los usuarios, hasta el punto de convertirse en indispensable su uso.


PowerPoint


Es un programa para realizar presentaciones, compatible con windows y macOS en las computadoras de escritorio y en dispositivos móviles con Android e iOS. Sus aplicaciones son útiles para el campo laboral y estudiantil.


El programa posee entre sus características plantillas predeterminadas, aunque el usuario puede diseñar las suyas; pueden insertarse imágenes; crear textos llamativos; efectos de transición para la presentación; efectos de animación para los elementos en las diapositivas; capacidad para insertar clips de audio y video; hipervínculos; entre otros.


One Note


Este software permite tomar notas de texto, a las que se le pueden agregar otros elementos complementarios como imágenes y elementos audiovisuales, pudiendo importar alguno de estos al programa desde otras aplicaciones. Estas notas pueden ser compartidas con otros usuarios.


Se puede utilizar en las PC (Windows, OS-X), además está disponible para la telefonía móvil (Windows Phone, Android, iOS). Es de gran utilidad para el ámbito escolar y permite apuntar en clases aun cuando no cuente con conexión en ese momento, pues puede sincronizarse luego.


Access


Es un grupo de programas para almacenar, extraer y modificar la información de una base de datos. Puede recoger datos desde otras aplicaciones de office con algún tema específico.


Para utilizar su última versión de 2013, el equipo debe tener ciertas características mínimas, como un procesador de 32 o 64 bits, una memoria RAM de al menos 1 gigabyte y 5 gigabytes de espacio en un disco.


Sharepoint


Es un grupo de productos y software que facilitan el trabajo de oficina. Este contiene programas que permiten navegar por Internet, administrar documentos, procesos y búsquedas. Puede descargarse de manera gratuita por usuarios de Windows Server, quienes recibirán funciones agregadas.


Entre las soluciones a disposición, se encuentran: windows SharePoint Services 3.0, Search Server 2008, Forms Server 2007, microsoft office SharePoint Server 2007 MOSS en sus versiones standard y enterprise, microsoft office groove server 2007 y microsoft office project server 2007.
