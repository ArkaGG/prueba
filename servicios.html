Otros Servicios


Skype


Es un programa gratuito de mensajería de texto, voz y video en línea. En el año 2013, Microsoft adquiere Windows Live Messenger, que se cohesiona con Skype, de manera que con el mismo usuario de WLM podía ingresar a Skype.


Este programa permite realizar llamadas a cualquier país a bajos costos; también puede asignarse un número desde un teléfono para llamar a usuarios conectados desde una computadora; y ofrece servicio de buzón de voz.


Microsoft security


Este Microsoft service o servicio de Microsoft, se trata de un antivirus gratuito para Windows para PC, que protege al sistema de virus, troyanos y espías. Anteriormente era Windows Defender. Este tipo de antivirus está diseñado para la protección de pequeñas empresas y usuarios domésticos, y es sencillo, ya que cuenta con tres colores como un semáforo: verde (libre de virus y protegido); amarillo (sin protección); y rojo (peligro para el equipo). Las bondades de este antivirus es que se actualiza de forma constante y automática.


Microsoft encarta


Era una enciclopedia virtual desarrollada en los años 90, que poseía un amplio contenido de interés general. Constaba de texto, imágenes, audios y videos, que complementaban el contenido, siendo su soporte físico un CD-ROM o DVD-ROM. Posteriormente salieron al mercado versiones con contenido actualizado, incluso con material web con la aparición del Internet y suscripciones anuales.


Internet explorer


Es un navegador de internet desarrollado para windows en 1995. Sus ediciones posteriores fueron el navegador predeterminado para este sistema, siendo uno de los más utilizados en 2003, mermando su popularidad con el paso de los años, al ser superado por sus competidores.


Este navegador todavía se mantiene como complemento para microsoft edge, navegador desarrollado por la compañía en 2015 para Windows 10 y Xbox One, llegando hasta la telefonía móvil Android e iOS en el 2017 y posteriormente en 2019 llega a macOS, sustituyendo al IE La última versión de IE es el 11, compatible con los sistemas Windows 7, 8, y 10.


Linkedin


Es una comunidad virtual destinada a la búsqueda de empleo, donde el usuario podrá subir un perfil con sus habilidades en el ámbito laboral para ofertar sus servicios profesionales a las empresas que forman parte de la red. Esta comunidad trabaja con la “Conexión”, que es la interrelación que tendrá un usuario con otros y con empresas. Estas pueden ser directas, de segundo grado (conexiones de las directas) y de tercer grado (conexiones de las de segundo grado).


Los usuarios pueden subir sus datos en forma de currículum vitae, puntualizando además sus experiencias laborales. Las empresas inscritas en la red, pueden publicar sus necesidades de recurso humano, ofertando sus puestos vacantes.


Cortana


Es un programa de asistencia virtual para varias versiones de Windows en smartphones y Xbox, abarcando muchas funciones para mejorar la experiencia del usuario. Entre ellas se encuentran la de realizar búsquedas por Bing, reconocimiento de voz o de música.


Entre sus funciones está el Cuaderno, que almacena información del usuario, mejorando su efectividad por patrones de preferencias; un sistema inteligente de recordatorios, que pueden asociarse a un contacto o ubicación específica; y, entre otras, puede resolver problemas matemáticos, realizar predicciones deportivas o determinar cambios de monedas.


Existen otra cantidad de servicios, aplicaciones y soluciones adicionales ofrecidas por la compañía, pero en este apartado se colocaron los más reconocidos en el mercado.
